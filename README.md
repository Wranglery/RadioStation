
简易版的小程序电台，实现在线加载节目功能，以及动态添加栏目功能。

小程序连接了后台程序，所以有部分功能需要搭建好后台从数据库读取数据才会显示，但是大部分页面是通过网络链接在线加载的。所以直接导入成小程序运行就可以了。
